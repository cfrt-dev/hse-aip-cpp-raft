◆ kv_apply()

int32_t kv_apply ( KvStore * store,
KvOp op,
const char * key,
const char * value )

Inserts, replaces or deletes a value in the in-memory key-value store.

Parameters
storeStore to mutate. The pointer must not be NULL.
opOperation code: KV_OP_SET writes a value, KV_OP_DEL removes a key.
keyNull-terminated key with length less than RAFT_MAX_KEY.
valueNull-terminated value used only for KV_OP_SET.
Returns
0 on success, negative value on invalid input or full store.
